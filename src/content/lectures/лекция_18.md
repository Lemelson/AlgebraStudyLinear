# Лекция 18. Прямые произведения, смежные классы, теоремы Лагранжа и Кэли

## Что связывает темы лекции

Прямое произведение строит большую группу из маленьких. Смежные классы, наоборот, разрезают группу на одинаковые блоки. Из этого разрезания получается теорема Лагранжа, а теорема Кэли показывает, что любую абстрактную группу можно понимать как группу перестановок.

## 1. Прямое произведение групп

Для групп \(G\) и \(H\) их **прямое произведение**
\[
G\times H=\{(g,h)\mid g\in G,\ h\in H\}
\]
имеет покоординатную операцию:
\[
(g_1,h_1)(g_2,h_2)=(g_1g_2,h_1h_2).
\]
Нейтральный элемент — \((e_G,e_H)\), а
\[
(g,h)^{-1}=(g^{-1},h^{-1}).
\]
Если группы конечны, то \(|G\times H|=|G||H|\). Порядок элемента:
\[
\operatorname{ord}(g,h)=
\operatorname{lcm}\bigl(\operatorname{ord}(g),\operatorname{ord}(h)\bigr)
\]
для конечных порядков координат.

### Почему \(\mathbb Z_2\times\mathbb Z_2\not\cong\mathbb Z_4\)

В обеих группах четыре элемента, но одного совпадения порядка группы недостаточно. В \(\mathbb Z_4\) есть элементы порядка \(4\): \(\overline1,\overline3\). В \(\mathbb Z_2\times\mathbb Z_2\) каждый ненулевой элемент имеет порядок \(2\):
\[
(a,b)+(a,b)=(0,0).
\]
Изоморфизм сохраняет порядок элементов, значит, группы не изоморфны.

\(\mathbb Z_2\times\mathbb Z_2\) называется **четверной группой Клейна** \(V_4\). Ее также можно увидеть внутри \(S_4\):
\[
V_4=\{e,(12)(34),(13)(24),(14)(23)\}.
\]

## 2. Левые смежные классы

Пусть \(H\le G\). Для \(g\in G\) **левый смежный класс** по \(H\) — множество
\[
gH=\{gh\mid h\in H\}.
\]
Это не обязательно подгруппа. Геометрическая интуиция: \(H\) — один шаблон, а умножение слева на \(g\) переносит этот шаблон в другое место группы.

Пример: в \((\mathbb Z,+)\) для \(H=3\mathbb Z\) смежные классы:
\[
0+H,\qquad1+H,\qquad2+H.
\]
Это числа с остатками \(0,1,2\) по модулю \(3\).

### Лемма 1. Классы либо совпадают, либо не пересекаются

**Формулировка.** Для любых \(a,b\in G\) либо \(aH=bH\), либо \(aH\cap bH=\varnothing\).

**Доказательство.** Предположим, классы пересекаются. Тогда существует \(x\), для которого
\[
x=ah_1=bh_2,\qquad h_1,h_2\in H.
\]
Отсюда
\[
a=bh_2h_1^{-1}.
\]
Так как \(h_2h_1^{-1}\in H\), имеем \(a\in bH\). Для любого \(ah\in aH\)
\[
ah=b(h_2h_1^{-1}h)\in bH,
\]
поэтому \(aH\subseteq bH\). Симметрично \(bH\subseteq aH\), следовательно, классы равны. \(\square\)

### Лемма 2. Смежный класс равномощен подгруппе

Отображение
\[
\varphi:H\to gH,\qquad \varphi(h)=gh
\]
биективно. Оно сюръективно по определению \(gH\). Если \(gh_1=gh_2\), умножение слева на \(g^{-1}\) дает \(h_1=h_2\), значит, оно инъективно. Поэтому \(|gH|=|H|\).

**Индекс подгруппы** \([G:H]\) — количество левых смежных классов \(H\) в \(G\).

## 3. Теорема Лагранжа

**Теорема.** Если \(G\) — конечная группа, \(H\le G\), то
\[
|G|=[G:H]\cdot|H|.
\]
В частности, \(|H|\mid|G|\).

**Доказательство.** Левые смежные классы покрывают \(G\), потому что каждый \(g\in G\) лежит в \(gH\): \(g=ge\). По лемме 1 разные классы не пересекаются, то есть дают разбиение \(G\). По лемме 2 каждый класс содержит \(|H|\) элементов. Классов \([G:H]\), поэтому общее число элементов равно их произведению. \(\square\)

### Следствие 1. Порядок элемента делит порядок группы

Подгруппа \(\langle a\rangle\) имеет \(\operatorname{ord}(a)\) элементов. По Лагранжу
\[
\operatorname{ord}(a)\mid|G|.
\]

### Следствие 2. \(a^{|G|}=e\)

Пусть \(|G|=m\operatorname{ord}(a)\). Тогда
\[
a^{|G|}=\left(a^{\operatorname{ord}(a)}\right)^m=e.
\]

### Следствие 3. Малая теорема Ферма

Для простого \(p\) ненулевые вычеты
\[
\mathbb Z_p^\times=\{\overline1,\ldots,\overline{p-1}\}
\]
образуют группу порядка \(p-1\). Поэтому для \(p\nmid a\)
\[
a^{p-1}\equiv1\pmod p.
\]
Умножая на \(a\), получаем форму, верную для любого целого \(a\):
\[
a^p\equiv a\pmod p.
\]

Теорема Лагранжа дает необходимое, но не достаточное условие существования подгруппы: из \(d\mid|G|\) не всегда следует, что в произвольной группе есть подгруппа порядка \(d\).

## 4. Теорема Кэли

**Теорема.** Любая группа \(G\) изоморфна некоторой подгруппе симметрической группы. Если \(|G|=n\), то \(G\) изоморфна подгруппе \(S_n\).

**Доказательство.**

Для каждого \(g\in G\) рассмотрим левый сдвиг
\[
L_g:G\to G,\qquad L_g(x)=gx.
\]
Это биекция: обратное отображение — \(L_{g^{-1}}\). Значит, \(L_g\) является перестановкой множества \(G\).

Определим
\[
\Phi:G\to\operatorname{Sym}(G),\qquad \Phi(g)=L_g.
\]
Проверим гомоморфность:
\[
\Phi(gh)(x)=L_{gh}(x)=(gh)x=g(hx)
=L_g(L_h(x)),
\]
поэтому \(\Phi(gh)=\Phi(g)\circ\Phi(h)\).

Проверим инъективность. Если \(L_g=L_h\), применим обе перестановки к \(e\):
\[
g=L_g(e)=L_h(e)=h.
\]
Следовательно, \(\Phi\) — инъективный гомоморфизм, а \(G\cong\Phi(G)\le\operatorname{Sym}(G)\). При \(|G|=n\) группа всех перестановок множества \(G\) изоморфна \(S_n\). \(\square\)

Смысл: каждый элемент группы можно представить как действие «умножить слева на этот элемент».

## Полностью разобранный пример

Пусть \(G=S_3\), \(H=\{e,(12)\}\). Найдем левые смежные классы и проверим Лагранжа.

1. \(|G|=6\), \(|H|=2\), поэтому ожидаем \(3\) класса.
2. Первый класс:
   \[
   eH=H=\{e,(12)\}.
   \]
3. Возьмем элемент \((13)\), еще не вошедший в класс:
   \[
   (13)H=\{(13),(13)(12)\}=\{(13),(123)\}.
   \]
   Здесь композицию читаем справа налево.
4. Остались \((23)\) и \((132)\):
   \[
   (23)H=\{(23),(23)(12)\}=\{(23),(132)\}.
   \]
5. Классы не пересекаются и вместе содержат все шесть элементов.
6. Индекс равен \(3\), и
   \[
   |S_3|=[S_3:H]|H|=3\cdot2=6.
   \]

Заметьте: левый класс \((13)H\) не подгруппа, потому что не содержит \(e\).

## Как применять теоремы в задачах

- Если нужно исключить порядок элемента, проверьте делимость \(\operatorname{ord}(a)\mid|G|\).
- Если надо подсчитать элементы в прямом произведении, сначала классифицируйте допустимые порядки каждой координаты.
- Если просят доказать Лагранжа, обязательно сформулируйте обе леммы о классах.
- В доказательстве Кэли не достаточно сказать «элемент действует перестановкой»: покажите биективность сдвига, гомоморфность и инъективность.

## Задачи для тренировки

### Задача 1

Сколько элементов порядка \(2\) в \(D_3\times S_3\times\mathbb Z_4\)?

**Подсказка.** Сначала посчитайте элементы, квадрат которых равен единице, включая единицу; из итогового произведения вычтите общую единицу.

**Ответ.** В \(D_3\cong S_3\) квадрат единицы равен единице у \(4\) элементов: \(e\) и трех отражений. В \(S_3\) — также у \(4\). В \(\mathbb Z_4\) у \(2\): \(\overline0,\overline2\). Поэтому квадрат равен единице у \(4\cdot4\cdot2=32\) элементов. Один из них — единица, остальные \(31\) имеют порядок \(2\).

### Задача 2

Может ли группа порядка \(35\) содержать элемент порядка \(6\)?

**Подсказка.** Используйте следствие из теоремы Лагранжа.

**Ответ.** Нет, потому что \(6\nmid35\).

### Задача 3

Докажите, что \(\mathbb Z_m\times\mathbb Z_n\) циклична при \(\gcd(m,n)=1\).

**Подсказка.** Найдите порядок \((\overline1,\overline1)\).

**Ответ.**
\[
\operatorname{ord}(\overline1,\overline1)
=\operatorname{lcm}(m,n)=mn=|\mathbb Z_m\times\mathbb Z_n|,
\]
поэтому этот элемент порождает всю группу.

## Частые ошибки

- Считать левый смежный класс подгруппой.
- Путать \(gH\) и \(Hg\) в неабелевой группе.
- В лемме о пересечении доказать только одно включение.
- Применять Лагранжа к бесконечной группе без дополнительного обоснования.
- Делать обратный вывод: «если \(d\mid|G|\), подгруппа порядка \(d\) обязательно есть».
- В прямом произведении складывать порядки координат вместо вычисления НОК.
- В доказательстве Кэли перепутать порядок композиции: \(L_g\circ L_h=L_{gh}\).
