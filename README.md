# Алгебра / Подготовка

Учебный сайт для последовательной подготовки к контрольной и экзамену по алгебре — от школьной базы до самостоятельного решения задач.

## Что работает

- два отдельных маршрута: шесть тем контрольной и восемь тем экзамена;
- полный материал первого коллоквиума: 56 определений, 16 доказательств, 14 основных задач и 19 задач Кострикина;
- 12 интерактивных визуализаций для тем контрольной — по две на каждую тему;
- официальный список из 92 определений;
- 34 вопроса с доказательством;
- 21 задача для подготовки с подсказками, полным решением и самопроверкой;
- 16 дополнительных вариантов — по два на каждую задачу реального экзамена;
- полная хронология 38 лекций курса ПИ 2025/2026, разделённая на четыре модуля;
- подробные LaTeX-уроки всех лекций 1–38: интуиция, доказательства, визуализации, примеры, задачи и самопроверка;
- 15 отдельных интерактивных SVG-схем для лекций первого семестра, каждая с несколькими состояниями и доступным описанием;
- экзаменационные метки P0/P1, официальные ссылки на записи и навигация между лекциями;
- все восемь подробных тематических глав: интуиция, определения, связи, пример, ошибки и практика;
- отдельная интерактивная визуализация в каждой теме;
- отдельные открываемые страницы каждого определения, доказательства и задачи;
- углублённые разборы и локальное сохранение прогресса.
- контекстный словарь из 104 терминов и 389 словоформ: задержанная подсказка при наведении, фокус с клавиатуры и открытие нажатием;
- LaTeX-формулировки и подробный первый разбор для всех 92 определений;
- аннотированные LaTeX-шаги с объяснением «зачем» для всех 34 доказательств.

## Запуск

```bash
npm install
npm run dev
```

Сборка и проверка:

```bash
npm run build
npm run lint
npm run test:control
npm run test:topics
npm run test:exam
npm run test:lectures
```

## Структура

- `src/data/controlCurriculum.js` — карта шести тем контрольной;
- `src/data/curriculum.js` — карта восьми экзаменационных тем;
- `src/data/lectures.js` — хронология, модули, приоритеты и официальные ссылки для 38 лекций;
- `src/content/lectures` — Markdown-уроки, которые рендерятся на сайте с формулами KaTeX;
- `src/pages` — страницы сайта;
- `src/components` — активные компоненты и визуализации;

Прогресс хранится локально в браузере. Регистрация и серверная синхронизация — отдельный следующий продуктовый этап.
